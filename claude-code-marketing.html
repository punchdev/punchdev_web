Introduction

Overview

Claude Code (claude.ai/code) is Anthropic's AI-powered CLI for software engineering tasks. Beyond code generation, it includes a robust ecosystem of marketing-focused skills and MCP (Model Context Protocol) integrations that enable direct automation of marketing and sales workflows.

This document outlines what's available out-of-the-box, what integrations exist, and practical automations you can implement today.

Integrations

MCP Integrations Available

MCP (Model Context Protocol) enables Claude Code to interact directly with external services. Your current setup includes Zapier-connected tools for Gmail, Google Calendar, Google Contacts, and Google Sheets.

Gmail

gmail_send_email — Send emails directly
gmail_create_draft — Create draft emails
gmail_reply_to_email — Auto-reply to emails
gmail_find_email — Search emails
gmail_create_draft_reply — Draft replies

Google Calendar

google_calendar_create_detailed_event — Create events with full details
google_calendar_quick_add_event — Quick event from text
google_calendar_find_event — Find events
google_calendar_update_event — Update events

Google Contacts

google_contacts_create_contact — Add contacts
google_contacts_find_contact — Find contacts
google_contacts_update_contact — Update contacts

Google Sheets

google_sheets_create_spreadsheet_row — Add rows
google_sheets_lookup_spreadsheet_row — Find data
google_sheets_update_spreadsheet_row — Update data

What You CANNOT Automate (Without Additional Tools)

LinkedIn Outreach

No LinkedIn MCP available. Requires LinkedIn API access or third-party tools. Use /copywriting for message templates and send manually.

Reddit Posting

No Reddit MCP. Requires Reddit API access for automated posting.

CRM Integration (Full)

Limited to Google Sheets for lead storage. HubSpot/Salesforce integration requires API keys or Zapier workflows.

Social Media Scheduling

Would need additional MCPs like Buffer or Hootsuite for scheduled posting.
